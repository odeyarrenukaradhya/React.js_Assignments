Smart Weather Forecast
==============================
Features Implemented:
- Real-time weather data for any city using OpenWeatherMap API
- Displays current temperature, humidity, and weather condition
- Shows weather icon based on condition
- Displays date, time, and timezone of searched city
- 5-day forecast with daily cards
- Search history with clickable cities (stored in localStorage)
- Current location weather using Geolocation API
- Temperature unit toggle (Celsius/Fahrenheit)
- Responsive design for desktop and mobile

APIs Used:
- OpenWeatherMap API (https://openweathermap.org/api)
- Geolocation API (browser native)

Note: Replace 'YOUR_OPENWEATHERMAP_API_KEY' in script.js with a valid API key from OpenWeatherMap.
