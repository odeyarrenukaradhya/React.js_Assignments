# Smart Task Manager

A simple React-based To-Do app that helps manage daily tasks with a clean interface.

## Features
- Add, delete, and mark tasks as complete or pending
- Persistent storage using localStorage
- Light/Dark theme toggle using Context API
- Controlled input with validation
- Input focus handled via useRef
- Task filtering (All, Completed, Pending)

## How To run:
1.Extract the folder
2.open the folder in VScode
3. Install dependencies using `npm install`
4. Start the project using `npm run dev`
