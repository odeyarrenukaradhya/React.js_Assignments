# About the Project
This project is aReact-based authentication interface that demonstrates the use of reusable components and children props. It provides a Login and Register interface with a professional, responsive, and visually appealing design.  
It allows users to:  
- Access a Login page with Email and Password fields  
- Access a Register page with Name, Email, Password, and Confirm Password fields  
- Switch between Login and Register pages using a simple tab navigation  
- See form validation messages
- Simulate form submissions using `alert`
The project demonstrates modular component design, dynamic props usage, and modern UI design with Tailwind CSS.

## Features
- Reusable Components: `Button`, `FormContainer`, `InputField`, `NavLink`  
- Children Props: All form fields and button contents are passed dynamically  
- Form Validation: Register page checks password confirmation and displays errors  
- Navigation: Tab-based switching between Login and Register pages  
- Responsive Design: Works on mobile, tablet, and desktop  
- Modern UI**: Clean card layout with hover and focus effects  
- Tailwind CSS Styling: Consistent, professional, and customizable UI

### Assumptions / Limitations
- Form submissions are simulated; no backend authentication
- Validation is limited to password confirmation on the Register page
