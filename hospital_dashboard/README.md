# Healthcare Patient Dashboard
This is a single-page application built with React to efficiently manage patient appointments and records within a healthcare setting. It demonstrates advanced use of React Hooks for state management, data optimization, and a clean, responsive UI.

## How to Run the Project
1. Downlaod and extract the zip folder and then named `hospital_dashboard`
2. Download the needed modules using `npm install`
3. Run the project using the `npm run dev`
