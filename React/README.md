# Personal Portfolio Website

A modern, responsive, and interactive **personal portfolio website** built using **React.js** and **Tailwind CSS**. The site showcases a developer’s profile, featured projects, and includes a working contact form — all designed with a clean, accessible, and minimal UI.

##  Features
-  Responsive design for desktop, tablet, and mobile.
- Smooth scrolling navigation (Home, About, Projects, Contact).
-  Project showcase with hover effects and animations.
- Dark mode toggle using Tailwind's dark classes.
- Functional contact form with basic validation and alert confirmation.
- sBuilt using modular React components and clean Tailwind utility classes.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons / ShadCN UI

## Sections Overview
- **Hero:** Introduction, profile image, and CTA buttons.
- **About:** Brief bio highlighting skills and experience.
- **Projects:** Interactive project cards with GitHub and demo links.
- **Contact:** Simple contact form and social links.
- **Footer:** Copyright and social handles.

## How to Run Locally

# Install dependencies
npm install

# Start development server
npm run dev
```
